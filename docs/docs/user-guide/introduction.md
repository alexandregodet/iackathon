# Bienvenue sur IAckathon

**IAckathon** est une application mobile Android qui vous permet de discuter avec une intelligence artificielle directement sur votre telephone, **sans connexion internet**.

## Qu'est-ce que IAckathon ?

IAckathon est un assistant conversationnel qui fonctionne entierement en local sur votre appareil. Contrairement aux assistants IA traditionnels comme ChatGPT ou Google Gemini qui necessitent une connexion internet, IAckathon telecharge le modele d'IA sur votre telephone et effectue tous les calculs localement.

### Avantages cles

| Caracteristique | Description |
|-----------------|-------------|
| **100% Hors ligne** | Une fois le modele telecharge, aucune connexion internet n'est necessaire |
| **Vie privee** | Vos conversations restent sur votre appareil, jamais envoyees a un serveur |
| **Gratuit** | Pas d'abonnement, pas de limites d'utilisation |
| **Rapide** | Reponses quasi-instantanees sans latence reseau |

## Fonctionnalites principales

### 1. Chat intelligent
Discutez naturellement avec l'IA pour obtenir des reponses a vos questions, de l'aide pour rediger des textes, ou simplement pour avoir une conversation.

### 2. Support d'images (Vision)
Avec les modeles multimodaux, vous pouvez envoyer des images et poser des questions a leur sujet. L'IA peut decrire, analyser et repondre a des questions sur le contenu visuel.

### 3. Documents PDF (RAG)
Importez vos documents PDF et posez des questions sur leur contenu. L'IA utilisera les informations du document pour vous repondre de maniere contextuelle.

### 4. Mode Reflexion (Thinking)
Certains modeles supportent un mode de reflexion avance qui montre le raisonnement de l'IA avant de donner sa reponse finale.

### 5. Modeles de prompts
Sauvegardez vos prompts favoris pour les reutiliser facilement dans vos conversations.

## Pour qui est IAckathon ?

- **Etudiants** : Aide aux devoirs, explications de concepts, revision
- **Professionnels** : Redaction, analyse de documents, brainstorming
- **Curieux** : Exploration de sujets, questions generales, conversations
- **Soucieux de la vie privee** : Discussions confidentielles sans envoi de donnees

## Modeles disponibles

IAckathon propose plusieurs modeles d'IA adaptes a differents besoins :

- **Gemma 3 1B** - Leger et rapide, ideal pour les appareils moins puissants
- **Gemma 3 Nano E2B/E4B** - Support vision (images), plus capable
- **DeepSeek R1 1.5B** - Mode reflexion avance pour des reponses plus reflechies
