# Premiers pas

Felicitations pour l'installation de IAckathon ! Ce guide vous accompagne dans vos premiers pas avec l'application.

## Ecran d'accueil

Au lancement de l'application, vous arrivez sur l'ecran d'accueil qui affiche :

- Le logo **IAckathon**
- Le bouton **SELECT_MODEL** pour choisir votre modele d'IA
- L'icone des parametres en haut a droite

## Etape 1 : Choisir un modele

Appuyez sur le bouton `> SELECT_MODEL` pour acceder a la selection des modeles.

### Types de modeles

Les modeles sont organises en deux categories :

#### Modeles multimodaux (MULTIMODAL)
Ces modeles peuvent analyser des images en plus du texte :
- **Gemma 3 Nano E2B** (2 Go) - Bon equilibre performance/taille
- **Gemma 3 Nano E4B** (4 Go) - Plus puissant, necessite plus de RAM

#### Modeles texte uniquement (TEXT_ONLY)
Ces modeles sont optimises pour le traitement du texte :
- **Gemma 3 1B** (900 Mo) - Leger, ideal pour debuter
- **DeepSeek R1 1.5B** (1.8 Go) - Mode reflexion avance

:::tip Conseil pour debuter
Si c'est votre premiere utilisation, nous recommandons **Gemma 3 1B** : il est leger, rapide a telecharger et fonctionne bien sur la plupart des appareils.
:::

## Etape 2 : Telecharger le modele

Apres avoir selectionne un modele, l'application verifie s'il est deja telecharge.

### Si le modele n'est pas telecharge

1. Un indicateur affiche la taille du telechargement
2. Appuyez sur le bouton de telechargement
3. Une barre de progression indique l'avancement
4. Attendez la fin du telechargement

:::warning Connexion Wi-Fi recommandee
Les modeles sont volumineux (900 Mo a 4 Go). Utilisez une connexion Wi-Fi pour eviter de consommer votre forfait mobile.
:::

### Si le modele est deja telecharge

L'application affiche "Modele installe" et vous pouvez directement le charger.

## Etape 3 : Charger le modele

Une fois telecharge, le modele doit etre charge en memoire :

1. Appuyez sur **Charger le modele**
2. Un indicateur de chargement s'affiche
3. Apres quelques secondes, vous arrivez sur l'ecran de chat

:::info Temps de chargement
Le chargement prend generalement 5 a 15 secondes selon votre appareil et la taille du modele.
:::

## Etape 4 : Votre premiere conversation

Vous etes maintenant pret a discuter avec l'IA !

### Envoyer un message

1. Tapez votre message dans le champ de saisie en bas de l'ecran
2. Appuyez sur la fleche pour envoyer
3. La reponse de l'IA s'affiche progressivement

### Exemples de premiers messages

Voici quelques idees pour commencer :

```
Bonjour ! Presente-toi brievement.
```

```
Explique-moi ce qu'est l'intelligence artificielle en termes simples.
```

```
Ecris un court poeme sur la nature.
```

## Navigation dans l'application

### Barre d'outils du chat

- **+** : Ajouter un document PDF ou une image, ou utiliser un modele de prompt
- **Fleche** : Envoyer le message
- **Historique** : Voir vos conversations precedentes

### Menu principal

- **Parametres** (icone engrenage) : Configurer l'application
- **Nouvelle conversation** : Demarrer une nouvelle discussion
